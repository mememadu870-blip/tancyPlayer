# tancyPlayer 产品改进计划（按优先级）

## 阶段 1：定时能力重构（高优先）
1. 将固定 15/30/60 分钟改为 `0~180` 分钟可滑动设置（Slider）。
2. 增加「启用定时」开关；关闭时计时器不生效。
3. 增加「播放完整歌曲后停止」复选项。
4. 启用后显示倒计时（设置页与播放器页同步显示）。
5. 触发规则：倒计时到 0 或当前曲播放结束（若勾选）时停止播放。

## 阶段 2：库筛选与搜索修复
1. 增加音频时长筛选（默认最短 30 秒，可配置），过滤铃声/短音频。
2. 扫描结果统一执行时长过滤。
3. 修复右上角搜索：支持按标题/歌手/专辑搜索。
4. 筛选配置持久化保存（SharedPreferences）。

## 阶段 3：页面体验优化
1. 移除 `Library / Playlists / Settings` 页面上浮播放条。
2. 保留底部导航，避免遮挡列表内容。

## 阶段 4：播放器可读性与交互增强
1. 播放器歌曲名改为滚动显示（支持长文件名完整可见）。
2. `Library / Playlists` 长按歌曲弹出详情（文件名、路径、时长、大小、专辑、歌手、hash 等）。

## 阶段 5：歌单功能补全
1. 点击歌单进入歌单详情页，显示歌单内歌曲。
2. 支持从歌单移除歌曲。
3. 支持删除歌单（含二次确认）。
4. 完善空歌单状态与返回流程。

## 阶段 6：回归与发布
1. 全流程回归：扫描→筛选→搜索→播放→定时→歌单增删→查重。
2. 修复构建 warning / 兼容问题。
3. 推送主分支并触发 GitHub Actions，输出 APK。
